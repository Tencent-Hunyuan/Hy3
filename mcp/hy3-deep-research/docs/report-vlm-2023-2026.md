## 执行摘要

2023–2026 年间，视觉语言模型（VLM）从任务特定的卷积–递归编码器–解码器，转向通用多模态大语言模型（MLLM），并在 2025–2026 年进一步分化为：**紧凑架构**、**多语言覆盖**、**感知中心统一推理**等方向。

核心创新（据现有综述证据）：开放域生成脱离封闭编解码器 [E4][E5]；transformer / mamba / hybrid 等紧凑设计 [E2]；跨语言视觉–语义对齐 [E3]；以及以 OpenAI O-series、DeepSeek R-series 等为信号的感知中心范式 [E1]。

贯穿始终的遗留问题：可靠视觉 grounding 与幻觉 [E4][E5]、语言中立 vs 文化适应张力 [E3]、感知范式系统理论仍不成熟 [E1]。

## 方法脉络与创新（主体）

### 代际 0：卷积–递归编码器–解码器（前 MLLM 基线）

- **思路**：CNN 提视觉特征 + RNN 逐词生成，封闭编解码器 [E4][E5]
- **创新标志**：确立视觉→语言监督映射基线
- **解决**：限定数据上的图像描述
- **遗留**：任务单一、难开放指令/推理、缺通用语义对齐

### 代际 1：MLLM 与开放生成（约 2023 起）

- **思路**：视觉信号接入 LLM 主干，开放域 caption / 指令跟随 / 零样本 [E4][E5]
- **相对前代创新**：抛弃独立卷积–循环流水线 → **统一多模态生成架构**；从封闭词表走向开放域
- **解决**：自由形式视觉语言生成，VLM 从专用模型转向通用助手
- **遗留**：**可靠 grounding 未解决**（幻觉物体、属性错绑、空间关系错误等）[E4][E5] —— 贯穿后续代际的主痛点

### 代际 2：效率与覆盖扩展（2025，两分支并行）

#### 分支 A：小型 VLM（sVLM）

- **思路**：资源受限下的紧凑多模态；架构分类含 transformer / mamba / hybrid [E2]
- **相对前代创新**：**高效紧凑架构**，引入非 Transformer 序列建模与混合设计，相对代际 1 的「大模型中心」
- **解决**：边缘 / 低资源部署
- **遗留**：性能折中与是否加剧幻觉 **待证实** [E2]

#### 分支 B：多语言 VLM

- **思路**：跨语言图文；encoder-only 与 generative；综述覆盖约 33 模型 / 23 基准 [E3]
- **相对前代创新**：单语（偏英语）→ **跨语言视觉语义对齐与生成**；显式讨论语言中立 vs 文化感知
- **解决**：非英语 VQA / 描述等地理扩展
- **遗留**：训练偏中立 → 与文化适应冲突，二者难兼顾 [E3]

### 代际 3：感知中心统一范式（2026）

- **思路**：以 O-series / R-series 等为标志，转向 perception-centric intelligence，统一理解与推理 [E1]
- **相对前代创新**：从「会描述/会对话」→ **感知为核心、与推理深度协同**（Structure → Synergy）
- **解决**：缓解早期 MLLM 感知偏浅、推理不足的问题（综述层归纳）
- **遗留**：系统 survey / 评测 / 理论框架仍稀缺，该代际尚新 [E1]

**脉络一句话**：监督映射基线 → 开放域 LLM 多模态生成 →（并行）紧凑化 + 多语言化 → 感知–推理协同；幻觉与 grounding 是跨代未解主线。

## 关键发现

1. Grounding / 幻觉从代际 1 至今无本质突破 [E4][E5]
2. 2025 创新突出「小」（sVLM）与「宽」（多语言）[E2][E3]
3. 2026 范式信号指向感知中心与推理统一，但体系未稳 [E1]
4. 多语言路线内含中立性–文化性矛盾 [E3]
