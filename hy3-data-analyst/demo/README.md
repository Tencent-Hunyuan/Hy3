# Demo Video

This directory contains the demonstration video for hy3-data-analyst.

## Video Content

The demo video (~1 minute) shows the complete data analysis workflow in two MCP clients (CodeBuddy and Trae):

1. **List files** — Open CodeBuddy, type "列出当前目录的数据文件" → Server returns the available CSV/JSON/Excel files.
2. **Statistical summary** — Type "分析 sales.csv" → Server returns Markdown-formatted statistics (shape, missing values, numeric summaries, top categories).
3. **Plot chart** — Type "画一个销量趋势图" → Server generates a line chart PNG and returns the image path with trend analysis.
4. **AI Q&A** — Type "哪个区域销售额最高？" → Server calls Hy3 API and returns an AI-generated answer with insights.
5. **Switch to Trae** — Repeat a similar workflow in Trae to demonstrate cross-client compatibility.
