腾讯混元 Hy3 是腾讯混元团队发布的一款大规模混合专家（MoE）语言模型。Hy3 总参数量达 295B，每次前向传播激活 21B 参数，并配备 3.8B 参数的 MTP 层。该模型在推理、Agent 任务和长上下文方面表现优异，支持 256K 上下文长度。

Hy3 采用 vLLM 和 SGLang 进行部署，提供 OpenAI 兼容的 API 接口。开发者可以通过 base_url 和 api_key 调用 Hy3 完成聊天、代码生成、数据分析等任务。Hy3 在多个产品场景中得到应用，包括办公、金融建模、前端开发和游戏开发。

为了降低使用门槛，Hy3 支持通过 TokenHub 平台获取 API Key。TokenHub 是腾讯混元模型的官方 API 平台，提供稳定、安全的模型调用服务。开发者只需配置环境变量即可在 Cursor、Cline、CodeBuddy 等 MCP 客户端中使用 Hy3。
