# Hy3 Quickstart

## 新增文件

- `quickstart/quickstart.md` - 快速入门文档，包含基础信息、最小可运行示例、参数说明和常见报错排查
- `quickstart/.env.example` - 环境变量配置示例
- `quickstart.py` - 快速使用测试
- `quickstart/examples/01_basic_chat/` - 基础对话示例（`README.md`、`basic_chat.py`、`basic_chat.ipynb`，单轮/多轮）
- `quickstart/examples/02_streaming/` - 流式请求示例（`README.md`、`streaming.py`、`streaming.ipynb`）
- `quickstart/examples/03_streaming_comparison/` - 流式 vs 非流式性能对比（`README.md`、`streaming_comparison.py`、`streaming_comparison.ipynb`）
- `quickstart/examples/04_tool_calling/` - 工具调用示例（`README.md`、`tool_calling.py`、`tool_calling.ipynb`，一次调用 + 多轮循环）
- `quickstart/examples/05_reasoning_mode/` - 思考模式示例（`README.md`、`reasoning_mode.py`、`reasoning_mode.ipynb`，开/关对比）
- `quickstart/examples/06_error_handling/` - 错误处理与重试示例（`README.md`、`error_handling.py`、`error_handling.ipynb`）
