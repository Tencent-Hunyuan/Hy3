RepoPilot Hy3 is a repository reproduction and error diagnosis assistant powered by the Hy3 API, with no training, fine-tuning, or local inference.
- **Repository Reproduction**: Upload a project zip or paste its README, dependencies, and configuration to generate an evidence-based run plan, risk analysis, and verification steps.
- **Error Diagnosis**: Combine project context with terminal logs to identify likely root causes and produce actionable repair commands.
- **Quick Deployment**: Use the FastAPI and React/Vite interface with one-click startup scripts for Windows, macOS, and Linux.
