# MindGraph AI

王凯犀牛鸟issue实战 "Build a vibe-coded application powered by Hy3"

基于腾讯混元 Hy3 API 的知识图谱 & 思维导图自动生成器。

## 功能

- **思维导图生成**：粘贴长文本或上传文件，调用 Hy3 生成分层思维导图，支持展开全部、缩放、导出 SVG/PDF
- **知识图谱生成**：输入关键词，调用 Hy3 生成实体关系图谱，支持节点下钻扩展、手动编辑、AI 建议补全、导出 SVG/PDF
