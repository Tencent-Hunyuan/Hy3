# Hy3 Quickstart Examples

Examples to get started with Hy3 API in 5 minutes.

## Prerequisites

- A running Hy3 API server (vLLM or SGLang)
- `pip install openai`

## Files

| File | What it does |
|------|-------------|
| `hello_hy3.py` | Basic chat, code generation, streaming |

## Run

```bash
export HY3_BASE_URL=http://127.0.0.1:8000/v1
export HY3_API_KEY=EMPTY
python hello_hy3.py
```
