# Example 3: Non-streaming vs Streaming — 首 token 时延 & 总耗时对比

## 说明

- **非流式**：TTFT ≈ 总耗时，适合后端处理
- **流式**：TTFT << 总耗时，适合实时展示

## 示例输出

```
============================================================
Non-streaming 模式
============================================================
首 token 时延 (TTFT): 3.21s  (非流式下 ≈ 总耗时)
总耗时:              3.21s
生成 tokens:          267
平均速度:            83.2 tokens/s

============================================================
Streaming 模式
============================================================
首 token 时延 (TTFT): 0.89s
总耗时:              3.45s
chunk 数量:            267
平均速度:            77.4 tokens/s

============================================================
对比总结
============================================================
指标                      Non-streaming          Streaming
───────────────────────── ────────────────────  ────────────────────
首 token 时延 (TTFT)       3.21                  0.89
总耗时                    3.21                  3.45
用户体验                  等待全部生成           实时输出
适用场景                  简单问答/分析          聊天/实时展示
```
