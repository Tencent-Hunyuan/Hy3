# Example 5: Reasoning Mode — 思考模式开/关对比

三种模式：

| 模式 | extra_body | 效果 |
|------|-----------|------|
| no_think | `{"reasoning_effort": "no_think"}` | 直接回答 |
| low | `{"reasoning_effort": "low"}` | 快速思维链 |
| high | `{"reasoning_effort": "high"}` | 完整思维链 |

## 示例输出

### no_think

```
设水池容量为 1 单位。
进水管速度 = 1/6 单位/小时
出水管速度 = 1/8 单位/小时
同时开时净速度 = 1/6 - 1/8 = 1/24 单位/小时
所需时间 = 1 ÷ (1/24) = 24 小时

答案：24 小时。
```

### low

```
<think>
进水管 6h 注满 → 每小时注 1/6
出水管 8h 排空 → 每小时排 1/8
净速度 = 1/6 - 1/8 = 4/24 - 3/24 = 1/24
时间 = 1 / (1/24) = 24
</think>
需要 24 小时。
```

### high

```
<think>
题目：水池进水管 6h 注满，出水管 8h 排空，同时开多久注满？

设水池总容量为 V。
进水管流量 = V/6（每小时）
出水管流量 = V/8（每小时）
同时开时净流入 = V/6 - V/8 = (4V-3V)/24 = V/24
需要的时间 t = V / (V/24) = 24h

验证：24h 进水 V/6 × 24 = 4V
       24h 出水 V/8 × 24 = 3V
       净存水 4V - 3V = V ✓
</think>
同时打开两个水管需要 **24 小时**才能将水池注满。
```
