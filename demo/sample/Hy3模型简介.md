# Hy3 模型简介

Hy3 是腾讯混元（Hunyuan）推出的大语言模型，定位为通用的对话与推理基础模型。

## 核心特点

- **架构**：采用 MoE（混合专家）结构，总参数约 295B，单次推理激活约 21B，兼顾效果与效率。
- **长上下文**：原生支持 256k 上下文窗口，可一次性处理长文档、代码库或会议记录。
- **兼容接口**：提供 OpenAI 兼容的 API（`/v1`），可直接复用现有 OpenAI SDK 接入。
- **推理模式**：支持 `reasoning_effort` 调节（如 `no_think` / `low` / `high`），按需平衡速度与深度。

## 适用场景

- 多文档问答与检索增强（RAG）
- 长文摘要、报告生成
- 代码理解与改写
- 多语种翻译与语气重写

## 接入方式

通过 `https://tokenhub.tencentmaas.com/v1` 调用，使用 Bearer Token 鉴权，模型名填 `hy3` 即可。
