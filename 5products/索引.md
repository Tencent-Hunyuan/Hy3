## 通用配置项参考

各工具的模型参数基本一致，可按需替换 Base URL：

| 配置项 | 填写内容 / 说明 |
| :--- | :--- |
| **API Base URL** | `https://openrouter.ai/api/v1`（或本地 vLLM/SGLang 地址） |
| **Model Name** | `hy3` / `tencent/hy3-295b-a21b`（依工具而定） |
| **API Key / Token** | 对应服务商获取的密钥 |
| **协议选择** | OpenAI Compatible / Chat Completion |

> **⚠️ 注意事项：** 长上下文建议 `Max Tokens`/`Context Window` 设为 8192+；代码生成 Temperature 0.2–0.4，创意写作 0.7–0.9；务必开启 **Stream 流式输出** 避免超时。
